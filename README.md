## 細項拆分

### 1) 場地（24,000）
1. 禮堂租金（主場）: 18,000
2. 佈場與撤場加時: 3,000
3. 清潔/管理費: 3,000

### 2) 物流與器材（12,000）
1. 比賽棋盤棋子（租/補充）: 4,500
2. 計時鐘（租）: 3,000
3. 桌椅及動線指示物料: 2,000
4. 搬運與車費: 2,500

### 3) 工作人員（16,000）
1. 裁判長（1）: 3,000
2. 裁判（4）: 6,000
3. 賽務台（2）: 3,000
4. 場務與接待（2）: 2,500
5. 攝影/記錄（1）: 1,500

### 4) 獎項與比賽金（18,000）
1. 公開組獎金: 8,000
2. 青少年組獎金: 4,000
3. 獎盃/獎牌: 3,000
4. 證書與紀念品: 3,000

### 5) 宣傳與報名系統（10,000）
1. 視覺設計（主視覺+社媒）: 3,000
2. 社交平台推廣: 4,000
3. 實體海報/單張: 1,500
4. 報名表單+客服/通知: 1,500

### 6) 保險與行政（6,000）
1. 公眾責任保險: 3,000
2. 行政文具/印刷: 2,000
3. 銀行/平台雜費: 1,000

### 7) 餐飲與補給（6,000）
1. 工作人員餐盒: 3,000
2. 水/簡單補給: 3,000

### 8) 預備金（8,000）
1. 場地超時與臨時支出: 4,000
2. 後備器材/緊急採購: 4,000
